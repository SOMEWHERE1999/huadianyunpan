# 已知限制

## 覆盖上传

不支持 Explorer“替换目标中的文件”、Copy-Item -Force 或 helper Overwrite。cgofuse 回调无法可靠提供 WinFsp create disposition/options；为避免普通同名复制误覆盖，现有目标默认拒绝。

## 在线编辑

不支持直接修改已有远端文件。编辑器可能使用截断、追加、临时文件和替换等不同流程，当前没有完整回滚与并发一致性保证。

## 同步、并发和上传

这不是实时双向同步盘；网页端变化不保证即时出现。多个客户端并发修改同一路径没有强一致性保证。不支持断点续传、分片和服务端 batch upload。目录和多文件上传由多次独立操作组成。

## 界面

没有独立 GUI、安装器或成熟启动脚本，主要使用 PowerShell 和资源管理器。
