# 功能列表

| 功能 | 状态 | 说明 |
|---|---|---|
| 目录浏览 | 支持 | 通过挂载盘浏览 |
| 文件读取/下载 | 支持 | 打开或复制到本地 |
| 新建目录 | 支持 | 真实远端操作 |
| 文件/目录重命名 | 支持 | 真实远端操作 |
| 文件/目录移动 | 支持 | 真实远端操作 |
| 文件及目录删除 | 支持 | 包括非空目录递归删除 |
| 单文件上传 | 支持 | 复制到新路径 |
| 多文件上传 | 支持 | 多次单文件上传 |
| 目录上传 | 支持 | 递归 mkdir 和上传 |
| 默认同名不覆盖 | 支持 | 冲突时安全拒绝 |
| AutoRename | 支持 | helper 生成新名称 |
| 覆盖上传 | 不支持 | 同名请跳过/取消 |
| 在线编辑 | 不支持 | 防止误修改远端 |
| 实时双向同步 | 不支持 | 无实时推送保证 |
| GUI | 不支持 | 使用资源管理器和命令行 |
| 断点/分片上传 | 不支持 | 当前为 staging upload |
