# 本地数据目录

本目录计划用于存放 daemon 运行时的本地持久化数据。发行包中仅含 `.gitkeep` 占位文件。

## 当前状态：数据库功能未实现

以下功能尚未开发，将在后续版本中补充：

- **SQLite 元数据库**（`metadata.db`）：缓存远端目录树和文件元信息，加速挂载盘浏览和属性查询。
- **同步任务队列**：upload/download/remove 任务的持久化存储，支持 daemon 重启后恢复未完成的任务。
- **同步根注册**：本地目录与远端路径的映射关系，用于实时同步模式下的目录监控和变更上传。
- **本地文件缓存**：已下载文件的本地副本管理，减少重复远端读取。
- **Staging 暂存区**：上传前本地临时文件目录（`staging/`），复制上传模式已使用此目录。

## 当前数据存储位置

由于数据库功能未实现，daemon 运行时数据由命令行参数 `--data-dir` 指定（默认 `%LOCALAPPDATA%\HuadianDrive`），而非使用本目录。启动脚本 `start.ps1` 也使用独立的数据目录。

## 安全提示

- `metadata.db` 不应包含 token、cookie、密码或授权头。
- 提交代码或打包分发时不应包含 `metadata.db`、staging 临时文件和本地缓存。
- 数据库文件建议放在本目录下统一管理。
